# Forsyth Schools Portal

Chrome Extension for Forsyth County Schools students — quick access to ClassLink, Infinite Campus, Canvas, and district resources.

## Quick install

Download the latest package from **[Releases → latest-build](https://github.com/jwatson2029/forsyth-schools-portal/releases/tag/latest-build)**, extract the zip, then load the folder in Chrome at `chrome://extensions` (Developer mode → Load unpacked).

## Repository layout

All extension source lives in [`forsyth-schools-portal/`](forsyth-schools-portal/). See that folder's [README](forsyth-schools-portal/README.md) for full documentation.

## Builds

Every push to `main` runs GitHub Actions and publishes an installable zip. Tagged releases (e.g. `v1.0.1`) create versioned releases automatically.
